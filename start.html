Video to ASCII Art Converter

Welcome!

This application converts videos to colored ASCII art using GPU acceleration.

Launch Converter System Test Create Test Video

Browser Compatibility

For best performance, use:

  • Chrome/Edge - Full WebGL2 support, MP4 export
  • Firefox - Good WebGL support, WebM export
  • Safari - Limited WebGL, may use fallback rendering

Note: If you see WebGL errors, the app will still work using Canvas fallback rendering.

Quick Start

  1. Click "Launch Converter" above
  2. Select a video file (MP4, WebM, etc.)
  3. Adjust settings:
    • Background Color - Choose ASCII background
    • Font Size - Smaller = more detail
    • Density - Character density
    • Color Accuracy - Color vs monochrome balance
  4. Click "Process Video" to see real-time conversion
  5. Click "Export as MP4" to save (exports as WebM in most browsers)

No Video File?

Click Create Test Video to generate a simple animated test video.

Troubleshooting

  • WebGL Error - Normal in some browsers, fallback rendering will be used
  • WASM Warning - Expected, using optimized JavaScript instead
  • Export Issues - Try different browsers, Chrome/Edge work best
  • Performance - Reduce video resolution or increase font size
